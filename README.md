## Installation âš™ï¸
### Prerequisites
- Node.js & npm
- MongoDB setup

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Pranjallpandey1504/closet.git
   cd closet
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up `.env` file:
   ```env
   MONGO_URI=your-mongodb-uri
   JWT_SECRET=your-secret-key
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open `http://localhost:3000` in your browser.
